Hash CSV File

Upload and hash a CSV file.

Input

Algorithm

Hashing algorithm to use. Defaults to the SHA-256 algorithm.

Column to Hash

Select which column to hash values. Defaults to column 1 only.

Identifier

Select which type of Id to send to Google. Adds a row at the top including this information. Defaults to PPID.

Delete Row

Include or Exclude the third column for delete. Defaults to Exclude.

Output

Not Started
